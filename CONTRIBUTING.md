# Contributing to AlphaBook

Thanks for your interest in AlphaBook! Pull requests are welcome.

## How to contribute

1. Fork the repository and create a branch from `main`.
2. Follow the setup guide in [SETUP.md](SETUP.md) to get a local instance running.
3. Keep changes focused — one feature or fix per pull request.
4. Open a pull request with a short description of what it changes and why.

For larger ideas (new games, strategy APIs, AI agent integration), please open an issue first so we can discuss the design before you invest time.
