🧮 Mental Math — Rules

🎯 How It Works

Answer timed math questions as fast as you can. The admin configures question types, difficulty, number of questions, and time per question. Answer before the timer runs out!

📊 Question Types

➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
📐 Exponents
⚖️ Comparison
🔢 Patterns
💯 Percentage

🏆 Scoring

Each correct answer = 1 point. If time runs out, the question is skipped (0 points). The player with the most correct answers wins!
